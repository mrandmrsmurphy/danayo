# Notes
- Note that this character is to be written in the Mainland/Japanese/Vietnamese way as [[Radical 053|广]] + [[林]], not 广+[[Radical 075|𣏟]], because of Shinjitai Rule.  The [[Stroke 11|stroke]] count is the same, as is the [[SKIP-3-3-8|SKIPcount]], however.
- Japanese [[Jōyō - Kōtō|high school]], [[Korean HS]], HSK #4, Dan'a'yo [[Grade 6]].
- The Chinese derived character 麼/麽 is forbidden.
- [[靡]] is a naming character.
### 借代字
- 魔 "witch, demon, mara", hence
	- 魔鬼 --> [[麻鬼]] = "demon", "monster"
	- 魔力 --> [[麻力]] = "magic power"
	- 魔女 --> [[麻女]] = "witch"
	- 惡魔 --> [[惡麻]] = "devil"
	- 魔羅 --> [[麻羅]] = "mara"
