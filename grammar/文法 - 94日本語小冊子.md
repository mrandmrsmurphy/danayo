# 単亜語入門（日本語話者向け）

## 一、単亜語とは何か

単亜語は、漢字を基礎とする分析型言語である。

特徴：
- 声調がない
- 一字一音節
- 連音・同化がない
- 動詞活用がない
- 基本語順は SVO（主語―動詞―目的語）
- 新字体を用いる

単亜語の新字体方針は、現代日本語の常用漢字とほぼ同じである。日本語話者にとって、この点で新たに覚えることはほとんどない。


## 二、音節構造
単亜語はモーラではなく、**音節**を単位とする。

一音節は次の形をとる：

子音＋（半母音）＋母音（＋子音）

すなわち：声母（頭子音）＋介音（省略可）＋核母音＋韻尾（省略可）。

各音節は独立して発音され、前後の音に影響されない。

### 声母（子音）

|符号|発音の目安|
|---|---|
|ㄅ|b に近い|
|ㄆ|p に近い|
|ㄇ|m に近い|
|ㄈ|f に近い|
|ㄉ|d に近い|
|ㄊ|t に近い|
|ㄋ|n に近い|
|ㄌ|l に近い|
|ㄐ|やや強めの j|
|ㄑ|やや強めの q|
|ㄙ|s に近い|
|ㄍ|g に近い|
|ㄎ|k に近い|
|ㄏ|h に近い|
|ㄨ|声母として用いる場合は w に近い|
|ㄧ|声母として用いる場合は y に近い|

説明：ㄫ（/ŋ/）は声母として使われず、音節末（韻尾）にのみ現れる。

### 母音

|符号|発音|
|---|---|
|ㄚ|a|
|ㄝ|e（純粋な e 音。「ə」ではない）|
|ㄧ|i|
|ㄛ|o|
|ㄨ|u|
|ㄜ|ə（弱化した過渡母音。外来語表記などに使用）|

単亜語の母音は日本語と同様に**基本母音**（カーディナル母音）を採用しており、日本語話者にとって発音しやすい体系である。

### 半母音
ㄧ・ㄨは音節内部で介音（半母音）としても機能し、それぞれ /j/・/w/ に対応する。声調とは無関係である。


## 三、声調はない

単亜語には声調の区別がない。

日本語のような高低アクセントとも無関係である。

すべての音節は同じ高さで発音してよい。

## 四、語末子音

### 1. -m / -n / -ng の区別

単亜語では
- -m
- -n
- -ng

は明確に区別される。

日本語の「ん」は一種類であるが、単亜語では三種類が独立した音として存在する。

したがって、すべてを同じ「ん」として処理しない。

### 2. -p / -t / -k について

単亜語には語末の
- -p
- -t
- -k

がある。

これは日本語の促音の「止め」に近い。

しかし、

- その後に母音は来ない
- 一音節で終わる
- 「隠れた母音」は存在しない

重要：

日本語では語末に子音だけで終わることはない。  
単亜語では可能である。

母音が消えるのではなく、最初から存在しない。

### 3. -i / -u について

単亜語には、高母音による滑音の韻尾 -i・-u もある。これらは二重母音のように働く。

## 五、基本文型

単亜語の基本語順は SVO である。

日本語：  
私は本を読む。

単亜語：  
我 閲読 書本

助詞は不要である。

語順が文法関係を決める。

## 六、時体標識

単亜語の時体標識の多くは動詞の**後**に置く（接尾辞）。「未」だけは動詞の**前**に置く（接頭辞）。

|字|機能|位置|
|---|---|---|
|已|完了|動詞の後（接尾辞）|
|中|進行|動詞の後（接尾辞）|
|将|未来|動詞の後（接尾辞）|
|未|未完了・否定完了|動詞の前（接頭辞）|

例：

我 食已  
我 食中  
我 食将  
我 未食

注意：「中」と「已」は併用でき、順序は固定で **中已**（進行が先、完了が後）となる。「将」が現れる場合は必ず動詞に最も近い位置に置く。

動詞自体は変化しない。

## 七、格助詞

単亜語には位置固定の格字がある。

|単亜語|日本語対応|
|---|---|
|之|の（属格、関係節にも用いる）|
|于|に|
|於|で／に|
|以|で|
|共|と|
|比|より|
|由|から|
|至|まで|
|似|のように|
|哉|よ|
|向|へ（方向）|
|様|に|

これらは必ず定位置に置く。

## 八、主題文と「是」

単亜語では主題構文が可能である。

ただし、述語が名詞の場合は必ず「是」を用いる。

正：

此書 是 好書

省略してはならない。

## 九、文末助詞

単亜語にも文末助詞がある。

例：

也（強調）  
耶（軽い疑問）  
乎（確認）  
哉（感嘆）  
了（完了）

文末に置き、順序は固定である。

## 十、要点整理

1. 声調なし
    
2. 一字一音節
    
3. モーラではなく音節
    
4. 語末子音がある
    
5. 活用なし
    
6. SVO語順
    
7. 名詞述語には「是」
    

単亜語は、日本語と同様に漢字文化圏に属するが、  
構造はより規則的で単純である。
