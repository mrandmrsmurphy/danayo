## Parts

1. Content Words - 実詞 - able to be topic or comment
	1. Nouns - 名詞 - need copulas to be comments
	2. Eventives - 事詞 - need an object (basically transitive verbs)
	3. Statives - 性詞 - expected to modify (like adjectives and stative verbs)
	4. Copula - 繋詞 - a closed set of six verbs
		1. 是 - to equal
		2. 非 - to not equal
		3. 有 - to have
		4. 無 - to not have
		5. 在 - to exist
		6. 莫 - to not exist
2. Function Words - 助詞 - attach to or stand-in for words
	1. Relationals - 関詞 - basically case endings
		1. Frame-forming (域化)
			1. 于 (animate dative)
			2. 於 (locative)
			3. 以 (instrumental)
			4. 共 (comitative)
			5. 比 (comparative)
			6. 由 (elative)
			7. 至 (terminaltive)
			8. 似 (similative)
			9. 哉 (vocative)
			10. 様 (adverbial)
		2. Topic-retaining (主題保持)
			1. 丈 (“only”)
			2. 且 (“also”)
			3. 公 (polite suffix)
	2. Modifiers - 修飾語 - qualifiers
		1. TAM 
			1. 已 - perfective
			2. 未 - negative perfective
			3. 将 - future
			4. 中 - continuous
		2. Deixis
			1. 此 - near
			2. 其 - middle
			3. 彼 - far
		3. Quantifiers
			1. 某 - some
			2. 皆 - all, every
			3. 多 - many
			4. 少 - few
			5. 全 - whole, entire
			6. 他 - another, additional
		4. Verbal
			1. 可 - able
			2. 不可 - unable
			3. 可不 - able to not
			4. 
		5. Ideophones - 擬詞 - a subgroup that only modifies comments and are rather evocative.  Some are so casual as to be 網語体 only.
	3. Correlatives - 代詞 - pronouns
		1. Correlative Pronoun - 對應代詞 - see table
		2. Personal Pronoun - 人称代詞 - namely:
			1. I - 我
			2. Thou - 君
			3. We - 我等
			4. You - 君等
			5. I (humble) ARCHAIC - 俺
			6. We (royal) - 朕
	4. Modals - 語気助詞 - Sentence Final Particles
		1. Dependent (依存) - Can be free-standing
		2. Interjections (自立) - 感詞 - These can be SFP or stand alone as a complete utterance
	5. Linkers - 連接詞 - conjunctions
3. Special Words - 特詞
	1. Numerals - 数詞 - make count phrases
	2. Classifiers - 量詞 - make count phrases
		1. People - 人, any person without reference to status, etc.
		2. Honorific - 名, deferential term for person, etc.
		3. Machines - 台, computers, cars, blenders, nanobots, etc.
		4. Animals - 匹, cat, elephants, mice, platypuses, etc.
		5. Books - 冊, books, magazines, newspapers, e-zines, etc.
		6. Cups - 杯, glass of water, shots of alcohol, etc.
		7. Flats - 枚, pizza, paper, plywood, etc.
		8. Long-Thins - 条, pencils, noodles, fingers, etc.
		9. Times - 段, seconds, days, years, millennia, etc.
		10. Events - 番, turns, happenings, repeats, etc.
		11. Bundles - 束, groups, bouquets, bales, classes, etc.
		12. Individual - 個, misc.
	3. Determiners - 限定詞 - used only in writing, not actually closed
		1. Names - 阿
		2. Amphibians - 蛙
		3. Birds - 鳥
		4. Sea-animals - 魚
		5. Creepy-crawlies - 虫
		6. Mammals - 獣
		7. Reptiles - 龍
		8. Flowers - 花
		9. Grasses - 草
		10. Trees - 木
		11. Clothes - 服
		12. Food - 飯
		13. Devices - 機
		14. Ideas - 義
		15. Structures - 構
