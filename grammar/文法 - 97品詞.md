## Parts

1. Content Words - <ruby>[実詞](words/実詞.md)<rt>ㄙㄧㄊㄙㄚ</rt></ruby> - able to be topic or comment
	1. Nouns - <ruby>[名詞](words/名詞.md)<rt>ㄇㄧㄫㄙㄚ</rt></ruby> - need copulas to be comments
		1. Ordinary Nouns - <ruby>[普通](words/普通.md)名詞<rt>ㄆㄛㄊㄛㄫㄇㄧㄫㄙㄚ</rt></ruby>
		2. Proper Nouns - <ruby>[固有](words/固有.md)名詞<rt>ㄍㄛ·⼜ㄇㄧㄫㄙㄚㄇㄧㄫㄙㄚ</rt></ruby> - Nouns + cannot take modification
		3. Adverbial Nouns - <ruby>[副用](words/副用.md)<rt>ㄆㄩㄎ·⼄ㄫ</rt></ruby><ruby>名詞<rt>ㄇㄧㄫㄙㄚ</rt></ruby> - able to be the frame with 0-derivation 
	2. Eventives - <ruby>[事詞](words/事詞.md)<rt>ㄐㄧㄙㄚ</rt></ruby> - need an object (basically transitive verbs)
	3. Statives - <ruby>[性詞](words/性詞.md)<rt>ㄙㄧㄫㄙㄚ</rt></ruby> - expected to modify (like adjectives and stative verbs, adnominal)
	4. Copula - <ruby>[系詞](words/系詞.md)<rt>ㄏㄝㄧㄙㄚ</rt></ruby> - a closed set of six verbs (<ruby>為<rt>⼔ㄋ</rt></ruby> and <ruby>即<rt>ㄐㄧㄎ</rt></ruby> are function words, <ruby>成<rt>ㄙㄧㄫ</rt></ruby> is 'become')
		1. <ruby>[是](words/是.md)<rt>ㄙㄝ</rt></ruby> - to equal
		2. <ruby>[非](words/非.md)<rt>ㄆㄧ</rt></ruby> - to not equal
		3. <ruby>[有](words/有.md)<rt>⼜</rt></ruby> - to have
		4. <ruby>[無](/words/無.md)<rt>ㄇㄜ</rt></ruby> - to not have
		5. <ruby>[在](/words/在.md)<rt>ㄐㄚㄧ</rt></ruby> - to exist
		6. <ruby>[莫](/words/莫.md)<rt>ㄇㄚㄎ</rt></ruby> - to not exist
2. Function Words - <ruby>[助詞](words/助詞.md)<rt>ㄐㄛㄙㄚ</rt></ruby> - attach to or stand-in for words
	1. Relationals - <ruby>[関詞](words/関詞.md)<rt>ㄍ⺢ㄇㄙㄚ</rt></ruby> - suffixing
		1. Frame-forming (<ruby>[域](words/域.md)<rt>ㄨㄧㄎ</rt></ruby><ruby>化<rt>ㄏ⺢</rt></ruby>) - **case particles - <ruby>[格助詞](words/格助詞.md)<rt>ㄍㄚㄎㄐㄛㄙㄚ</rt></ruby>**
			1. Genitive - <ruby>[之](/words/之.md)<rt>ㄊㄧ</rt></ruby> ([[属格]])
			2. Dative - <ruby>[于](/words/于.md)<rt>ㄨ</rt></ruby> ([[有生]] <ruby>[与格](words/与格.md)<rt>⼄ㄍㄚㄎ</rt></ruby>)
			3. Locative - <ruby>[於](words/於.md)<rt>ㄛ</rt></ruby> ([[処格]])
			4. Instrumental - <ruby>[以](words/以.md)<rt>ㄧ</rt></ruby> ([[具格]])
			5. Comitative - <ruby>[共](/words/共.md)<rt>ㄍ⼄ㄫ</rt></ruby> ([[共格]])
			6. Comparative - <ruby>[比](/words/比.md)<rt>ㄅㄧㄜ</rt></ruby> ([[比較格]])
			7. Ablative - <ruby>[由](/words/由.md)<rt>⼜ㄛ</rt></ruby> ([[奪格]])
			8. Terminative - <ruby>[至](/words/至.md)<rt>ㄐㄧㄜ</rt></ruby> ([[終止格]])
			9. Similative - <ruby>[似](/words/似.md)<rt>ㄙㄚ</rt></ruby> ([[類似格]])
			10. Vocative - <ruby>[哉](/words/哉.md)<rt>ㄐㄚㄧ</rt></ruby> ([[呼格]])
			11. Allative - <ruby>[向](/words/向.md)<rt>ㄏ⼘ㄫ</rt></ruby> ([[向格]])
			12. Adverbial - <ruby>[様](/words/様.md)<rt>⼘ㄫ</rt></ruby> ([[様態格]])
		2. Topic-retaining (<ruby>[主題](words/主題.md)<rt>ㄐㄨㄊㄝㄧ</rt></ruby><ruby>[保持](words/保持.md)<rt>ㄅㄚㄨㄉㄧ</rt></ruby>)
			1. Exclusive -<ruby>[只](/words/只.md)<rt>ㄑㄝ</rt></ruby>
			2. Additional -<ruby>[且](/words/且.md)<rt>ㄑㄛ</rt></ruby>
			3. Politeness suffix - <ruby>[公](/words/公.md)<rt>ㄍㄛㄫ</rt></ruby>
			4. Activer agent -<ruby>[者](/words/者.md)<rt>ㄑㄚ</rt></ruby>
			5. Passive agent -<ruby>[被](/words/被.md)<rt>ㄆㄧ</rt></ruby>
			6. Object -<ruby>[物](/words/物.md)<rt>ㄚㄨㄊ</rt></ruby>
			7. Action -<ruby>[事](words/事.md)<rt>ㄐㄧ</rt></ruby>
	2. Modifiers - [[修飾語]]
		1. "Tense" (<ruby>[時制](/words/時制.md)<rt>ㄙㄧㄐㄝㄧ</rt></ruby>)
			1. <ruby>[已](/words/已.md)<rt>ㄧ</rt></ruby> - perfective
			2. <ruby>[未](/words/未.md)<rt>ㄇㄨㄧ</rt></ruby> - negative perfective
			3. <ruby>[将](/words/将.md)<rt>ㄐ⺢ㄫ</rt></ruby> - future
			4. <ruby>[中](/words/中.md)<rt>ㄐㄨㄫ</rt></ruby> - continuous
		2. Demonstratives - [[指示詞]]
			1. <ruby>[此](/words/此.md)<rt>ㄑㄜ</rt></ruby> - near
			2. <ruby>[其](/words/其.md)<rt>ㄍㄜ</rt></ruby> - middle
			3. <ruby>[彼](/words/彼.md)<rt>ㄅㄜ</rt></ruby> - far
		3. Quantifiers - [[量化詞]]
			1. <ruby>[某](/words/某.md)<rt>ㄇㄛㄨ</rt></ruby> - some
			2. <ruby>[皆](/words/皆.md)<rt>ㄍ⼶</rt></ruby> - all, every
			3. <ruby>[多](/words/多.md)<rt>ㄉㄚ</rt></ruby> - many
			4. <ruby>[少](/words/少.md)<rt>ㄙㄛㄨ</rt></ruby> - few
			5. <ruby>[全](/words/全.md)<rt>ㄐ⼔ㄋ</rt></ruby> - whole, entire
			6. <ruby>[他](/words/他.md)<rt>ㄊㄜ</rt></ruby> - another, additional
		4. Moods - <ruby>[情態](/words/情態.md)<rt>ㄑㄧㄫㄊㄚㄧ</rt></ruby> ***prefixing***
			1. <ruby>[可](/words/可.md)<rt>ㄎㄜ</rt></ruby> - able
			2. <ruby>[不可](/words/不可.md)<rt>ㄅㄛㄊㄎㄜ</rt></ruby> - unable
			3. [[可不]] - able to not
			4. [[不可不]] - unable to not
			5. <ruby>[不](/words/不.md)<rt>ㄅㄛㄊ</rt></ruby> - not
			6. [[可以]] - permissive
			7. [[不可以]] - impermissive
		5. Ideophones - [[擬詞]] - a subgroup that only modifies comments and are typically the result of reduplication.  Some are so casual as to be [[網語体]] only.
	3. Pronouns - <ruby>[代詞](words/代詞.md)<rt>ㄉㄚㄧㄙㄚ</rt></ruby>
		1. Correlative Pronouns - <ruby>[対応](words/対応.md)<rt>ㄉㄛㄧ·ㄧㄫ</rt></ruby><ruby>代詞<rt>ㄉㄚㄧㄙㄚ</rt></ruby> - see table
		2. Personal Pronouns - <ruby>[人称](words/人称.md)<rt>ㄋㄧㄋㄑㄧㄫ</rt></ruby><ruby>代詞<rt>ㄉㄚㄧㄙㄚ</rt></ruby> - namely:
			1. I - <ruby>[我](words/我.md)<rt>ㄚ</rt></ruby>
			2. Thou - <ruby>[君](/words/君.md)<rt>ㄍㄨㄋ</rt></ruby>
			3. We - <ruby>[我等](words/我等.md)<rt>ㄚㄉㄨㄫ</rt></ruby>
			4. You - <ruby>[君等](words/君等.md)<rt>ㄍㄨㄋㄉㄨㄫ</rt></ruby>
			5. I (humble) *ARCHAIC* - [[俺]]
			6. We (royal) - [[../words/朕]]
			7. You (intimate) - [[汝]]
			8. They *ARCHAIC* - [[厥]]
	4. Modals - <ruby>[語気](words/語気.md)<rt>·⼄ㄎㄧㄜ</rt></ruby><ruby>[助詞](words/助詞.md)<rt>ㄐㄛㄙㄚ</rt></ruby> - Sentence Final Particles
		1. Dependent ([[依存]]) - Can be free-standing
		2. Interjections ([[自立]]) - [[感詞]] - These can be SFP or stand alone as a complete utterance
	5. Linkers - <ruby>[連接詞](words/連接詞.md)<rt>ㄌ⼶ㄋㄐㄛㄆㄙㄚ</rt></ruby> - conjunctions
		1. <ruby>[而](/words/而.md)<rt>ㄋㄧ</rt></ruby>
		2. <ruby>[与](/words/与.md)<rt>⼄</rt></ruby>
3. Special Words - <ruby>[特詞](words/特詞.md)<rt>ㄉㄨㄎㄙㄚ</rt></ruby>
	1. Numerals - <ruby>[数詞](words/数詞.md)<rt>ㄙㄨㄙㄚ</rt></ruby> - make count phrases
		1. One, Two, Three, Four, Five, six, Seven, Eight, Nine = 
		2. Ten, [[百]], <ruby>[一千](/words/一千.md)<rt>ㄧㄊㄑㄝㄋ</rt></ruby>, <ruby>[一万](/words/一万.md)<rt>ㄧㄊㄇㄛㄋ</rt></ruby>,  <ruby>[一億](/words/一億.md)<rt>ㄧㄊ·ㄧㄎ</rt></ruby>, <ruby>[一兆](/words/一兆.md)<rt>ㄧㄊㄐㄚㄨ</rt></ruby>, 溝
		3. Half - <ruby>[一半](/words/一半.md)<rt>ㄧㄊㄅㄚㄋ</rt></ruby>
	2. Classifiers - <ruby>[量詞](words/量詞.md)<rt>ㄌ⼘ㄫㄙㄚ</rt></ruby> - make count phrases
		1. People - <ruby>[人](words/人.md)<rt>ㄋㄧㄋ</rt></ruby>, any person without reference to status, etc.
		2. Honorific - <ruby>[名](words/名.md)<rt>ㄇㄧㄫ</rt></ruby>, deferential term for person, etc.
		3. Machines - <ruby>[台](/words/台.md)<rt>ㄉㄚㄧ</rt></ruby>, computers, cars, blenders, nano-bots, etc.
		4. Animals - <ruby>[匹](/words/匹.md)<rt>ㄆㄧㄊ</rt></ruby>, cat, elephants, mice, platypuses, etc.
		5. Books - <ruby>[冊](/words/冊.md)<rt>ㄑㄚㄎ</rt></ruby>, books, magazines, newspapers, e-zines, etc.
		6. Cups - <ruby>[杯](/words/杯.md)<rt>ㄅㄛㄧ</rt></ruby>, glass of water, shots of alcohol, etc.
		7. Flats - <ruby>[枚](/words/枚.md)<rt>ㄇㄛㄧ</rt></ruby>, pizza, paper, plywood, etc.
		8. Long-Thins - <ruby>[条](/words/条.md)<rt>ㄐㄛ</rt></ruby>, pencils, noodles, fingers, etc.
		9. Times - <ruby>[段](words/段.md)<rt>ㄉ⺢ㄋ</rt></ruby>, seconds, days, years, millennia, etc.
		10. Events - <ruby>[番](/words/番.md)<rt>ㄈㄚㄋ</rt></ruby>, turns, happenings, repeats, etc.
		11. Bundles - <ruby>[束](/words/束.md)<rt>ㄙ⼄ㄎ</rt></ruby>, groups, bouquets, bales, classes, etc.
		12. Individual - <ruby>[個](words/個.md)<rt>ㄍㄚ</rt></ruby>, misc.
	3. Determiners - <ruby>[限定詞](words/限定詞.md)<rt>ㄏㄚㄋㄐㄝㄫㄙㄚ</rt></ruby> - used only in writing, not actually closed, not necessarily words (can be name-only \*characters)
		1. Names - <ruby>[阿](words/阿.md)<rt>ㄚ</rt></ruby>

		7. Reptiles - [龍](../words/龍.md)
