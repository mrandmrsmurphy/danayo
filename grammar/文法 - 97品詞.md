## Parts

1. Content Words - <ruby>[実詞](words/実詞.md)<rt>ㄙㄧㄊㄙㄚ</rt></ruby> - able to be topic or comment
	1. Nouns - <ruby>[名詞](words/名詞.md)<rt>ㄇㄧㄥㄙㄚ</rt></ruby> - need copulas to be comments
		1. Ordinary Nouns - <ruby>[普通](words/普通.md)名詞<rt>ㄆㄛㄊㄛㄥ</rt></ruby>
		2. Proper Nouns - <ruby>[固有](words/固有.md)名詞<rt>ㄍㄛ·˙ㄨㄇㄧㄥㄙㄚ</rt></ruby> - Nouns + cannot take modification
		3. Adverbial Nouns - <ruby>[副用](words/副用.md)<rt>ㄆㄩㄎ·˙ㄛㄥ</rt></ruby><ruby>名詞<rt>ㄇㄧㄥㄙㄚ</rt></ruby> - able to be the frame with 0-derivation 
	2. Eventives - <ruby>[事詞](words/事詞.md)<rt>ㄐㄧㄙㄚ</rt></ruby> - need an object (basically transitive verbs)
	3. Statives - <ruby>[性詞](words/性詞.md)<rt>ㄙㄧㄥㄙㄚ</rt></ruby> - expected to modify (like adjectives and stative verbs, adnominal)
	4. Copula - <ruby>[系詞](words/系詞.md)<rt>ㄏㄝ˙ㄙㄚ</rt></ruby> - a closed set of six verbs (<ruby>為<rt>ˇㄝㄋ</rt></ruby> and <ruby>即<rt>ㄐㄧㄎ</rt></ruby> are function words, <ruby>成<rt>ㄙㄧㄥ</rt></ruby> is 'become')
		1. <ruby>[是](words/是.md)<rt>ㄙㄝ</rt></ruby> - to equal
		2. <ruby>[非](words/非.md)<rt>ㄆㄧ</rt></ruby> - to not equal
		3. <ruby>[有](words/有.md)<rt>˙ㄨ</rt></ruby> - to have
		4. [[無]] - to not have
		5. [[../words/在]] - to exist
		6. [[../words/莫]] - to not exist
2. Function Words - <ruby>[助詞](words/助詞.md)<rt>ㄐㄛㄙㄚ</rt></ruby> - attach to or stand-in for words
	1. Relationals - <ruby>[関詞](words/関詞.md)<rt>ㄍˇㄚㄇㄙㄚ</rt></ruby> - suffixing
		1. Frame-forming (<ruby>[域](words/域.md)<rt>ˇㄧㄎ</rt></ruby><ruby>化<rt>ㄏˇㄚ</rt></ruby>) - **case particles - <ruby>[格助詞](words/格助詞.md)<rt>ㄍㄚㄎㄐㄛㄙㄚ</rt></ruby>**
			1. Dative - [[../words/于]] ([[有生]] <ruby>[与格](words/与格.md)<rt>˙ㄛㄍㄚㄎ</rt></ruby>)
			2. Locative - <ruby>[於](words/於.md)<rt>ㄛ</rt></ruby> ([[処格]])
			3. Instrumental - <ruby>[以](words/以.md)<rt>ㄧ</rt></ruby> ([[具格]])
			4. Comitative - [[共]] ([[共格]])
			5. Comparative - [[比]] ([[比較格]])
			6. Ablative - [[由]] ([[奪格]])
			7. Terminative - [[至]] ([[終止格]])
			8. Similative - [[words/似]] ([[類似格]])
			9. Vocative - [[哉]] ([[呼格]])
			10. Allative - [[向]] ([[向格]])
			11. Genitive - [[之]] ([[属格]])
			12. Adverbial - [[様]] ([[様態格]])
		2. Topic-retaining (<ruby>[主題](words/主題.md)<rt>ㄐㄨㄊㄝ˙</rt></ruby><ruby>[保持](words/保持.md)<rt>ㄅㄚˇㄉㄧ</rt></ruby>)
			1. Exclusive -<ruby>[只](/words/只.md)<rt>ㄑㄝ</rt></ruby>
			2. Additional -<ruby>[且](/words/且.md)<rt>ㄑㄛ</rt></ruby>
			3. Polite - [[公]]
			4. Activer agent -<ruby>[者](/words/者.md)<rt>ㄑㄚ</rt></ruby>
			5. Passive agent -<ruby>[被](/words/被.md)<rt>ㄆㄧ</rt></ruby>
			6. Object -<ruby>[物](/words/物.md)<rt>ㄚㄨㄊ</rt></ruby>
			7. Action -<ruby>[事](words/事.md)<rt>ㄐㄧ</rt></ruby>
	2. Modifiers - [[修飾語]]
		1. "Tense" ([[時制]])
			1. <ruby>[已](/words/已.md)<rt>ㄧ</rt></ruby> - perfective
			2. [[未]] - negative perfective
			3. <ruby>[将](/words/将.md)<rt>ㄐˇㄚㄥ</rt></ruby> - future
			4. <ruby>[中](/words/中.md)<rt>ㄐㄨㄥ</rt></ruby> - continuous
		2. Demonstratives - [[指示詞]]
			1. <ruby>[此](/words/此.md)<rt>ㄑㄝ</rt></ruby> - near
			2. <ruby>[其](/words/其.md)<rt>ㄍㄧ</rt></ruby> - middle
			3. <ruby>[彼](/words/彼.md)<rt>ㄅㄧ</rt></ruby> - far
		3. Quantifiers - [[量化詞]]
			1. [[../words/某]] - some
			2. [[../words/皆]] - all, every
			3. [[多]] - many
			4. [[少]] - few
			5. [[全]] - whole, entire
			6. [[words/他]] - another, additional
		4. Moods - [[情態]]
			1. [[可]] - able
			2. [[不可]] - unable
			3. [[可不]] - able to not
			4. [[不可不]] - unable to not
			5. [[不]] - not
			6. [[可以]] - permissive
			7. [[不可以]] - impermissive
		5. Ideophones - [[擬詞]] - a subgroup that only modifies comments and are typically the result of reduplication.  Some are so casual as to be [[網語体]] only.
	3. Pronouns - <ruby>[代詞](words/代詞.md)<rt>ㄉㄚ˙ㄙㄚ</rt></ruby>
		1. Correlative Pronouns - <ruby>[対応](words/対応.md)<rt>ㄉㄛ˙·ㄧㄥ</rt></ruby><ruby>代詞<rt>ㄉㄚ˙ㄙㄚ</rt></ruby> - see table
		2. Personal Pronouns - <ruby>[人称](words/人称.md)<rt>ㄋㄧㄋㄑㄧㄥ</rt></ruby><ruby>代詞<rt>ㄉㄚ˙ㄙㄚ</rt></ruby> - namely:
			1. I - <ruby>[我](words/我.md)<rt>ㄚ</rt></ruby>
			2. Thou - [[君]]
			3. We - <ruby>[我等](words/我等.md)<rt>ㄚㄉㄨㄥ</rt></ruby>
			4. You - <ruby>[君等](words/君等.md)<rt>ㄍㄨㄋㄉㄨㄥ</rt></ruby>
			5. I (humble) *ARCHAIC* - [[俺]]
			6. We (royal) - [[朕]]
			7. You (intimate) - [[汝]]
	4. Modals - <ruby>[語気](words/語気.md)<rt>·˙ㄛㄎㄧ˙</rt></ruby><ruby>[助詞](words/助詞.md)<rt>ㄐㄛㄙㄚ</rt></ruby> - Sentence Final Particles
		1. Dependent ([[依存]]) - Can be free-standing
		2. Interjections ([[自立]]) - [[感詞]] - These can be SFP or stand alone as a complete utterance
	5. Linkers - <ruby>[連接詞](words/連接詞.md)<rt>ㄌ˙ㄝㄋㄐㄛㄆㄙㄚ</rt></ruby> - conjunctions
3. Special Words - <ruby>[特詞](words/特詞.md)<rt>ㄉㄨㄎㄙㄚ</rt></ruby>
	1. Numerals - <ruby>[数詞](words/数詞.md)<rt>ㄙㄨㄙㄚ</rt></ruby> - make count phrases
		1. 一, Two, Three, Four, Five, six, Seven, Eight, Nine, 
		2. Ten, 百, Thousand, 万,  億, 兆, 溝
		3. Half - 半
	2. Classifiers - <ruby>[量詞](words/量詞.md)<rt>ㄌ˙ㄚㄥㄙㄚ</rt></ruby> - make count phrases
		1. People - <ruby>[人](words/人.md)<rt>ㄋㄧㄋ</rt></ruby>, any person without reference to status, etc.
		2. Honorific - <ruby>[名](words/名.md)<rt>ㄇㄧㄥ</rt></ruby>, deferential term for person, etc.
		3. Machines - [[台]], computers, cars, blenders, nano-bots, etc.
		4. Animals - [[匹]], cat, elephants, mice, platypuses, etc.
		5. Books - [[冊]], books, magazines, newspapers, e-zines, etc.
		6. Cups - [[杯]], glass of water, shots of alcohol, etc.
		7. Flats - [[枚]], pizza, paper, plywood, etc.
		8. Long-Thins - [[条]], pencils, noodles, fingers, etc.
		9. Times - <ruby>[段](words/段.md)<rt>ㄉˇㄚㄋ</rt></ruby>, seconds, days, years, millennia, etc.
		10. Events - [[番]], turns, happenings, repeats, etc.
		11. Bundles - [[束]], groups, bouquets, bales, classes, etc.
		12. Individual - <ruby>[個](words/個.md)<rt>ㄍㄚ</rt></ruby>, misc.
	3. Determiners - <ruby>[限定詞](words/限定詞.md)<rt>ㄏㄚㄋㄐㄝㄥㄙㄚ</rt></ruby> - used only in writing, not actually closed, not necessarily words
		1. Names - <ruby>[阿](words/阿.md)<rt>ㄚ</rt></ruby>
		2. Amphibians - <ruby>[蛙](characters/蛙.md)<rt>ˇㄚ</rt></ruby>
		3. Birds - [[鳥]]
		4. Sea-animals - <ruby>[魚](words/魚.md)<rt>˙ㄛ</rt></rt></ruby>
		5. Creepy-crawlies - <ruby>[虫](characters/虫.md)<rt>ㄐㄨㄥ</rt></ruby>
		6. Mammals - [[獣]]
		7. Reptiles - [[龍]]
		8. Flowers - [[花]]
		9. Grasses - [[草]]
		10. Trees - [[words/木]]
		11. Clothes - [[服]]
		12. Food - [[飯]]
		13. Devices - [[機]]
		14. Ideas - [[義]]
		15. Structures - [[構]]
