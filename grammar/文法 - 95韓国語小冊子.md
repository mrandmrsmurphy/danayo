# 단아요 빠른 시작 안내 (한국어 화자용)

## 1. 단아요는 어떤 언어인가?
단아요는 다음과 같은 특징을 가진 언어이다.
- 굴절이 없다. (활용, 어미 변화 없음)    
- 존댓말 단계가 없다.
- 조사가 거의 없다.
- 음운 변화가 없다. (연음, 동화, 된소리화 없음)
- 기본 어순은 **SVO** (주어-동사-목적어)

한국어보다 문법 구조는 훨씬 단순하다.

## 2. 발음 체계 (보포모포 기반)
단아요는 보포모포(주음부호)를 사용한다.  
그러나 성조는 없으며, 기존 성조 기호는 **반모음 표시**로 사용한다.

### 2-1. 초성
|주음|한국어 대응|주의|
|---|---|---|
|ㄅ ㄆ ㄇ|ㅂ ㅍ ㅁ|된소리 아님|
|ㄉ ㄊ ㄋ ㄌ|ㄷ ㅌ ㄴ ㄹ|된소리 아님|
|ㄐ ㄑ ㄙ|ㅈ ㅊ ㅅ|ㅉ 아님|
|ㄍ ㄎ ㄏ|ㄱ ㅋ ㅎ|ㄲ 아님|

- ㄅ ㄉ ㄍ ㄐ 은 약간 유성에 가깝게 발음해도 좋다.  
- 절대로 된소리화하지 않는다.

### 2-2. 모음

|주음|한국어|
|---|---|
|ㄚ|아|
|ㄝ|에|
|ㄧ|이|
|ㄛ|오|
|ㄨ|우|

모음 체계는 매우 단순하다.

### 2-3. 반모음 (성조 기호의 새로운 역할)
단아요에는 성조가 없다.

대신 성조 기호는 반모음을 표시한다.
- ˙ = /y/ 계열 (이 반모음)
- ˇ = /w/ 계열 (우 반모음)

예:
- ㄍㄚ˙ = 갸 (가 + 이)
- ㄍ˙ㄚˇ = gyaw (갸 + 우)
- ㄙˇㄝㄊ = swet (쉗 비슷)

초기 한글에서 개가 가+이로 분석되었던 것과 비슷한 구조이다.

### 2-4. 종성 (받침)

|주음|한국어 받침|
|---|---|
|ㄆ ㄊ ㄎ|ㅂ ㄷ ㄱ|
|ㄇ ㄋ ㄥ|ㅁ ㄴ ㅇ|

- 모두 **받침처럼 불파음**으로 발음한다.
- 그러나 **연음, 동화, 된소리화가 절대 일어나지 않는다.**
- 각 음절은 완전히 독립적으로 발음한다.

예:  
각 음절이 서로 영향을 주지 않는다.

## 3. 기본 문장 구조
단아요는 **SVO 구조**이다.

한국어:  
나는 책을 읽는다.

단아요:  
<ruby>我<rt>ㄚ</rt></ruby> <ruby>閲読<rt>˙ㄝㄊㄉㄛㄎ</rt></ruby> <ruby>書本<rt>ㄙ˙ㄛㄅㄛㄋ</rt></ruby>。

한국어식으로 읽으면:  
아 열독 서본

특징:
- 은/는 없음
- 이/가 없음
- 을/를 없음
- 동사 활용 없음

그냥 순서로 의미를 구분한다.

## 4. 격 표현 (한국어 대응)
단아요는 한자 형태의 격 표지를 사용하지만 기능은 한국어 조사와 매우 비슷하다.

|단아요|기능|한국어 대응|
|---|---|---|
|于|여격|에게|
|於|처격|에 / 에서|
|以|도구격|로 / 으로|
|共|공동격|와 / 과|
|比|비교격|보다|
|由|출발격|에서|
|至|도달격|까지|
|似|유사격|처럼|
|哉|호격|아 / 야|
|様|상태|-게|

예:

<ruby>君<rt>ㄍㄨㄋ</rt></ruby> <ruby>父親<rt>ㄅㄨㄑㄧㄋ</rt></ruby><ruby>于<rt>ㄨ</rt></ruby> <ruby>贈与<rt>ㄐㄨㄥ·˙ㄛ</rt></ruby> <ruby>書本<rt>ㄙ˙ㄛㄅㄛㄋ</rt></ruby>。
너는 아버지에게 책을 증여했다.

|---|---|---|

## 6. 핵심 원칙 정리
1. 음절은 서로 영향을 주지 않는다.
2. 된소리 없음.
3. 활용 없음.
4. SVO 유지.
5. 조사 없이 순서로 의미 구분.

단아요는  
“활용 없는 한자어 중심의 단순 문장 구조”라고 생각하면 이해하기 쉽다.
