## Datacheck
```base
version: 1
views:
  - type: table
    name: Outgoing links
    filters:
      and:
        - this.file.hasLink(file)
    order:
      - file.name
      - hsk_level
      - grade_level
    sort:
      - property: grade_level
        direction: ASC
      - property: hsk_level
        direction: ASC
    columns:
      - file
      - file.path
      - file.links.length
```
