> **Final 覺** evolved into ㄚㄎ, with 4 homophony-driven exceptions on ㄛㄎ (see below)

## CJKV Evolution
覺 [ɣʌk] lands 18 of 22 characters on **ㄚㄎ**. The remaining 4 — 角, 剝, 擢, 渥 — all shift to **ㄛㄎ** instead, and this is the cleanest homophony-avoidance case found on this project so far: every single one of the four is dodging a same-initial+same-final slot that was already occupied.

- 角 (would regularly be ㄍㄚㄎ) dodges a slot already shared by 較 and 覚.
- 剝 (would regularly be ㄅㄚㄎ) dodges 駁, which already holds that slot.
- 擢 (would regularly be ㄐㄚㄎ) dodges 捉, which already holds that slot.
- 渥 (would regularly be ㄚㄎ, null-initial) dodges a slot already shared by 岳 and 握.

No daughter-language reading distinguishes any of these four from their regular-outcome neighbors — the split is purely about avoiding a crowded syllable, not phonological conditioning. All four exceptions shift in the same direction (ㄚ → ㄛ), which may itself be a convenient, reusable "overflow vowel" for this coda.
