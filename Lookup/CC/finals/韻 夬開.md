## CJKV Evolution
夬開 [ɣai] is fully uniform: all 5 characters land on **ㄚㄧ**, with no w-glide — exactly as expected for an 開 (unrounded) final, per the 合→w-glide rule confirmed across every other 開/合 pair checked so far.

Worth flagging: the Vowels table lists the *same* documented D value (`wai`) for this final and its rounded sibling [[韻 夬合|夬合]] — but this page's real corpus outcome (ㄚㄧ, no w) matches every other 開 final's behavior, not a `wai`-type outcome. This looks like a copy-paste duplication in the reference table rather than a genuine dual match, similar to the [[韻 刪合|刪合]] table-discrepancy flagged earlier in this sweep — worth a second look at the table itself, though outside the scope of this page.
