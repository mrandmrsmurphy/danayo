# 単亜語入門（粤语使用者）

## 一、単亜語是什么？
単亜語是一种以汉字为核心的分析型语言。

特点：
- 没有声调
- 每个字只有一个音节
- 没有连读变化
- 没有动词变位
- 基本语序为 SVO（主语－动词－宾语）
- 使用新字体

例：読（讀）、国（國）、単（單）

## 二、发音原则
### 1. 没有声调
単亜語没有声调对立。
可以用粤语的音节感觉去读，但不要加入高低升降的声调区别。

### 2. 音节结构
単亜語保留完整的入声结构：
- -p
- -t
- -k
- -m
- -n
- -ng

所有字：
- 不连读
- 不变调
- 不同化
- 每个音节独立发音

## 三、基本句型
単亜語采用 SVO 结构。

粤语：  
我食饭。

単亜語：  
我 食 米飯

没有主语标记  
没有宾语标记  
没有动词变化

语序决定语法关系。

## 四、时体标记
単亜語有固定的时体字，放在动词前。

|标记|功能|
|---|---|
|已|完成|
|未|未完成／尚未|
|将|将来|
|中|进行|

例：

我 已食  
＝ 我食咗

我 中食  
＝ 我食紧

我 将食  
＝ 我会食

我 未食  
＝ 我未食

## 五、格助词
単亜語使用固定位置的格字。

|単亜語|功能|
|---|---|
|于|向、给|
|於|在|
|以|用|
|共|同|
|比|比较|
|由|从|
|至|到|
|似|像|
|哉|呼格|
|様|方式|

例：

我 于 父 贈 書  
＝ 我送书俾阿爸

## 六、主题句规则
単亜語允许主题句。

但如果述语是名词，必须使用「是」。

正确：

此書 是 好書

不可省略「是」。

## 七、句末助词
単亜語也有句末助词，但位置固定，不可随意叠加。

例：

也（强调）  
耶（轻问）  
乎（反问／确认）  
哉（感叹）  
了（完成）

句末助词放在句尾，不可更换顺序。

## 八、核心规则总结
1. 没有声调
2. 每字一音
3. 不连读
4. 语序固定
5. 动词不变
6. 名词作述语必须用「是」

对粤语使用者而言，単亜語在音节结构上非常接近，但语法更加规则、更加固定。
