**조/jo/ㄐㄛ** is a very popular [[Syllables|syllable]] in polysyllablic words.  Alone, it is a classifier for LONG-THIN objects (条).  See before for other usages.

- [[条 (char)]] LONG-THIN classifier [[条]] 
- [[祖]] "ancestor" but requires <ruby>[[祖先]]<rt>ㄐㄛㄙㄝㄋ</rt></ruby>
- [[助]] but requires [[援助]]
- [[組]] but requires [[組合]]
- [[貯]] but requires [[貯蔵]]
- [[租]] but requires [[租金]]
- [[煮]] but requires [[煮沸]]
- [[粗]] but requires [[粗糙]]
