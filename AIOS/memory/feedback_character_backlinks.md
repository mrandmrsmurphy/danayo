When perfecting or creating a word, immediately update each constituent character page's `## Words` section to include the new word. This is **part of the word-perfection task**, not a separate follow-up. Format:

```markdown
## Words
- <ruby>[[WORD]]<rt>注音</rt></ruby> "english gloss"
```

Create the `## Words` section if it doesn't exist; place it after `## Notes`, before `## Chengyu`.

**Why:** `## Words` criterion 9 for a character's `date-last-perfect` requires that every Dan'a'yo word using that character appears there with a gloss. If this step is skipped, an entire session's worth of word work leaves character pages with uncertifiable backlinks — requiring a separate retroactive audit. This was caught in session after perfecting 15 words (汝, 或, 終止格, 擬詞, 感詞, 但, 若, 雖, 故而, 而且, 然後, 系詞, 数字, 代詞, 人称) without updating any character pages.

**How to apply:** At the end of every word-perfection task — before marking it done — open every constituent character file and add the entry. This is step 5 of the word creation skill and is in the pre-flight checklist. Do not defer it.
