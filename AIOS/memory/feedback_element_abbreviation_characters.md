description: A character's "abbreviation for [element]" note is a permanent fact about the character itself, not incidental Words-section trivia
type: feedback
---

When a character page's Notes section says (or once said) "abbreviation for [element]," that fact belongs to the character itself, not just to the one coined word that uses it.

**Why:** This vault's periodic-table naming system assigns individual Dan'a'yo characters to function as an element's own symbol/abbreviation — the same role a letter like "Gd," "Sg," or "Cf" plays in a real periodic table — not merely as one ingredient of a single compound word. During a character-perfecting `/loop` sweep, three such notes (多 → gadolinium, 西 → seaborgium, 里 → curium) were mistakenly treated as duplicate/incidental content and folded into the `## Words` entry for the coined element word, in one case (西) losing the "abbreviation" framing entirely down to a bare gloss. The user caught this and had all three corrected.

**How to apply:** Whenever a character's Notes contains (or has ever contained) an "abbreviation for X" statement, keep it as its own dedicated trailing Notes bullet — phrased so it's clear the character itself functions as that element's symbol — and link to the actual coined word (e.g. `— see [[加多金]]`). Do this even when the same word already has its own `## Words` entry; the two facts are not redundant (one documents the character's special abbreviation role, the other documents the compound word's meaning), so don't discard the Notes-level fact just because a Words-level entry already exists for the same compound.
