**라** is a Dan'a'yo syllable meaning "to strip, undress" (裸).  For other uses, see below.

### Forms
*Roman*: la; *IPA*: /ɾa/
### Characters
- [[裸]] - "to strip, undress, be naked"
#### Advanced
* [[蘿]] - "radish", but requires <ruby>[[蘿蔔]]<rt>라복</rt></ruby>
* [[螺]] - "spiral, screw", but requires <ruby>[[螺旋]]<rt>라숸</rt></ruby>
* [[鑼]] - "gong", but requires <ruby>[[銅鑼]]<rt>라동</rt></ruby>

#### Name
* [[邏]] - (name) "patrol, watch" but name only
* [[喇]] - (name) 'rain sound' but name only
