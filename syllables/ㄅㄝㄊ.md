> **ㄅㄝㄊ** means "don't" () {or Pelodiscus sinensis ()}.

## Characters
```dataviewjs
const pages = dv.pages()
  .where(p => p.注音 === "ㄅㄝㄊ")

const titles = pages
  .map(p => p.file.name)
  .sort()                // optional: alphabetical sort
  .join(", ");

dv.paragraph(titles || "No matching pages");
```
