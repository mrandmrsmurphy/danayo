**·ㄚ/아/'a** is a Dan'a'yo [[Syllables|syllable]] which means "I/me" (我) or the ideophone "ah" (阿).  In necessary situations it can mean "moth" (蛾).  For other uses in compounds, see below.
## Characters
### Common
- [[我]] I, me
- [[阿]] ah, {determiner for names}
- [[亜]] Asia, secondary; but requires <ruby>亜細亜<rt>·ㄚㄙㄝㄨ·ㄚ</rt></ruby>
- [[牙]] fang, but requires <ruby>長牙<rt>ㄐㄚㄥ·ㄚ</rt></ruby>
- [[芽]] bud, but requires <ruby>新芽<rt>ㄙㄧㄋ·ㄚ</rt></ruby>
- [[餓]] hungry, but requires <ruby>飢餓<rt>ㄍㄧㄧ·ㄚ</rt></ruby>
- [[雅]] graceful, but requires <ruby>典雅<rt>ㄉㄝㄋ·ㄚ</rt></ruby>
### Advanced
- [[唖]] mute, but requires 啞口
- [[俄]] suddenly, but requires <ruby>俄然<rt>·ㄚㄋㄧㄝㄋ</rt></ruby>
- [[峨]] lofty, but requires <ruby>峨峨<rt>·ㄚ·ㄚ</rt></ruby>
### Naming
- [[蛾]] moth
- [[鵝]] domestic goose, but requires <ruby>鵝鳥<rt>·ㄚㄑㄛㄨ</rt></ruby>
- [[吾]] I, but we use 我 instead
- [[娥]] beautiful, but name only
