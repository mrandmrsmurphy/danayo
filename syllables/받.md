**받** is a Dan'a'yo syllable meaning "eight" (八) or "bowl" (鉢).  For other uses see below.

### Forms
*Roman*: bad; *IPA*: /bat/
### Characters
- [[八 (char)]] "eight", the word [[八]]
- [[髪]] "hair", but requires <ruby>[[頭髪]]<rt>톳받</rt></ruby>
- [[鉢]] "bowl"
- [[抜]] "select, choose as best", but requires <ruby>[[選抜]]<rt>숸받</rt></ruby>

#### Advanced
* [[捌]] "to disentangle"
* [[叭]] "trumpet, horn", but requires <ruby>[[喇叭]]<rt>라받</rt></ruby>

#### Name
* [[犮]] "shape of a running dog", but name only
