## Notes
- [阻](../characters/阻.md) "thwart, block" + [碍](../characters/碍.md) "obstruct, hinder" — to obstruct, hinder, block. HSK6-level Mandarin vocabulary, also attested in Cantonese. Japanese instead uses the parallel formation 阻害 (そがい), with a different second character (害, "harm"); not to be confused with Korean 저애 (沮礙), which uses a different first character (沮) entirely.
