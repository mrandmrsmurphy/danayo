## Notes
- [分 (char)](../characters/分%20(char).md) "divide" + [裂](../characters/裂.md) "split; crack"

**Real `korean` bug caught, the same standing North Korean rule as [[卵子]]'s 란자 fix earlier this sweep**: the stored value `분열` applies South Korean 두음법칙-style regularization of 렬→열; `characters/裂.md`'s own field is `렬`, and North Korean 문화어 orthography preserves 분렬 (not 분열) for this exact compound (verified via search — this is a well-documented, named North/South spelling difference, not an obscure edge case). Corrected `분열`→`분렬`.

Mandarin fēnliè, Cantonese fan1 lit6, and Japanese ぶんれつ (bunretsu) are all the standard word for "to split, divide, fracture" (人格分裂, "split personality"; 細胞分裂, "cell division"). Vietnamese phân liệt is a real, attested Sino-Vietnamese compound (verified via search), though the everyday native word for splitting/dividing generally is simply chia (or chia rẽ for a rift/division between people).
