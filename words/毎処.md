## Notes
- [毎 (char)](../characters/毎%20(char).md) "every; each; distributive" + [処 (char)](../characters/処%20(char).md) "place; location"

毎処 is the distributive place correlative — "at each place, per location, in every spot one by one." As a 副用名詞 (adverbial noun), it frames the following predicate as applying to each location individually, distributing responsibility or action across places rather than treating them as a collective whole: 毎処、安全確認を行う ("at each location, conduct a safety check"); 毎処に標識を立てる ("erect a sign at each location"). This parallels the distributive function of 毎日 (every day), 毎回 (each time), and 毎人 (per person) — the 毎 prefix in each case signals individualised, one-by-one application.

No source language has a single word that precisely matches 毎処. Mandarin 每處 (měi chù, "at each place") is the closest compositional equivalent; 各地 (gèdì, "each locale/region") captures a similar distributive-spatial sense in a more idiomatic form. Japanese uses 場所ごと (basho goto, "per place, location by location") or 各地で (kakuchi de, "at each region") depending on context; neither is a single compact pronoun. Korean 곳마다 (got-mada, "at each place" — using the distributive suffix -마다 as in 사람마다) is the most parallel form. These circumlocutions in the source languages make 毎処 one of the cleaner additions that the Dan'a'yo correlative system provides: a single compact form for the distributive-place concept.

毎処 contrasts with 皆処 (everywhere, all locations collectively) exactly as 毎人 contrasts with 皆人: 皆処は危険だ ("everywhere is dangerous" — a collective statement about all places) vs 毎処を確認する ("check each location one by one" — a distributive instruction). The spatial row (処) and temporal row (時) exhibit this same distributive/collective contrast through 毎処/皆処 and 毎時/皆時, giving Dan'a'yo compact forms for "at each place" and "at each time" alongside "everywhere" and "always."
