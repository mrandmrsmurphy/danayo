## Notes
**Form.** 欧金 is [欧](characters/欧.md) + [金 (char)](characters/金%20(char).md) “metal,” following the rule for lanthanides.

**Motivation.** The international name _europium_ is derived from **Europe**. In CJKV, Europe is **欧洲**, so 欧 serves as the conventional one-character abbreviation.

**Type of formation.**  
A **toponymic abbreviation**:

- source: 欧洲 (Europe)
- reduction: 欧
- suffixation: 金

This is fully parallel to the other place-derived elements (法素, 波素, 露金, 莱金, 丹金).

**Fit within the system.**

1. Preserves the naming origin without phonetic borrowing.
2. Uses a standard, widely recognized abbreviation.
3. Maintains one-character uniqueness for symbol use.
4. Uses 金 appropriately for a metal.

**Comparative CJKV forms.**

- Modern Chinese: 铕 (phonetic)
- Japanese: ユウロピウム (phonetic)
- Korean: 유로퓸 (phonetic)
- Vietnamese: europi (phonetic)

Modern usage is phonetic; 欧金 restores a graph-based geographic reference.

**Clarity and issues.**

- 欧 also appears in other contexts (e.g., 欧州, 欧文), but consistently signals “Europe.”
- Within a chemical register, reserving 欧 as an element symbol removes ambiguity.

**Conclusion.**  
欧金 is a straightforward and well-formed toponymic name, consistent with the system and with established CJKV abbreviation practice.
