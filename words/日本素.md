## Notes
- [日 (char)](../characters/日%20(char).md) "sun" + [本 (char)](../characters/本%20(char).md) "root, origin" + [素](../characters/素.md) "element" — together, [[日本]] "Japan" + the element suffix.

Unlike most of this row, 日本素 is a calque rather than a sound transliteration: element 113 was discovered at RIKEN and named *nihonium* directly after Japan (Nihon), so this entry translates the etymology itself rather than transliterating the IUPAC syllables — the same logic as [[茜素]] (rubidium), which calques the Latin root instead of transliterating it. Every other source language instead borrows the IUPAC name wholesale as a loanword (Japanese ニホニウム, Korean 니호늄), making this one of the few elements in the table with no real-world CJKV precedent to follow or split the difference between.
