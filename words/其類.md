## Notes
- [其 (char)](../characters/其%20(char).md) "that; medial demonstrative; its/their (classical)" + [類](../characters/類.md) "kind; sort; category; type"

其類 is the medial kind/category correlative — "that kind; that sort of thing; the type we've been discussing." Like all 其-column entries, it picks up a category from the shared discourse context or the addressee's frame: 其類の話はもう止めよう ("let's stop discussing that sort of thing"); 其類の問題には慣れている ("I'm used to problems of that kind"). The category is understood from what has just been said or implied — 其類 points to a type already in the conversation.

Japanese そういう (sō iu, "that kind of; of that sort") or そのような (sono yō na, "like that; of that nature") are the medial adnominal equivalents. Korean 그런 종류 (geureon jongryu, "that kind — medial") or 그런 alone as an adnominal, the medial member of the 이런/그런/저런 deictic kind-triad. Mandarin 那種 (nàzhǒng, "that kind") covers both medial and distal, since Mandarin has a binary rather than three-way deictic system for categories.

The original 其類 file carried the Korean value 이런 종류 — the proximal "this kind," which belongs to 此類. The error placed the medial slot with the proximal form. The correct medial form is 그런 (종류), completing the Korean 이런/그런/저런 triad that maps cleanly onto Dan'a'yo's 此/其/彼 columns across all three deictic rows (人, 処, 類, and others) that use this pattern.
