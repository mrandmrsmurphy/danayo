## Notes
- [不 (char)](../characters/不%20(char).md) "not" + [用](../characters/用.md) "use"

不用 is "not necessary; unused; useless" — a straightforward negation of 用 "use," covering both the adverbial sense (不用謝, "no need to thank me/you're welcome") and the adjectival sense (an unused, discarded item).

Mandarin 不用 (bùyòng, "no need to; needn't" — 不用客氣 "no need to stand on ceremony"). Cantonese 不用 (bat1jung6). Japanese 不用 (ふよう, fuyō) leans toward the concrete "unused, no longer needed" sense (不用品 fuyōhin, "unwanted/discarded items"), distinct from the near-homophone 不要 (fuyō — same reading, different characters) which instead means "unnecessary" in a more abstract sense; the two collide in speech and are disambiguated only by context or writing. Korean 불용 (bullyong) patterns the same way as the concrete "unused, disused" sense (불용품, "unused goods") — 不 here takes its more common reading 불 rather than 부, following the general Sino-Korean rule that 不 reads 부 mainly before ㄷ/ㅈ-initial syllables and 불 otherwise. Vietnamese does not appear to calque 不用 directly with the same two characters — the closest everyday equivalents, bất tất ("unnecessary," built with 必 rather than 用) and vô dụng ("useless," built with 無 rather than 不), draw on different character pairings for the same semantic territory, so no `vietnamese` value is given here rather than guessing one.
