## Notes

**Form.** 隠金 is 隠 “hidden; concealed” + 金 “metal,” following the rule for transition metals (here extended to the f-block).

**Motivation.** The international name _lanthanum_ derives from Greek λανθάνειν (_lanthanein_) “to lie hidden, to escape notice.” The element was so named because it was difficult to detect, being concealed within cerium minerals. 隠 is a direct and economical rendering of this core idea.

**Type of formation.**  
A **straight semantic calque** of the Greek verb root:

- “to be hidden” → 隠

This is one of the cleaner cases where the Indo-European etymology maps directly onto a high-frequency Sinitic graph.

**Fit within the system.**

1. Fully meaning-based, no phonetic compromise.
2. Uses 金 appropriately for a metal in the extended d/f classification.
3. Provides a simple, unambiguous, high-frequency symbol.
4. Aligns well with neighboring actinide/lanthanide naming logic (e.g., 始金 for the start of a series).

**Comparative CJKV forms.**

- Modern Chinese: 镧 (phonetic)
- Japanese: ランタン (phonetic)
- Korean: 란타넘 (phonetic)
- Vietnamese: lantan (phonetic)

Modern forms are phonetic; 隠金 restores the original semantic content.

**Clarity and nuance.**

- 隠 is broad (“hide, conceal”), but this matches the Greek well.
- It also subtly reflects the chemical behavior of lanthanides as a “hidden series,” though that is a secondary resonance.

**Conclusion.**  
隠金 is an exact and well-formed semantic reconstruction of _lanthanum_, fitting into the system with minimal compromise and high transparency.
