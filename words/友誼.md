## Notes
- [友](../characters/友.md) "friend" + [誼](../characters/誼.md) "friendship" — `characters/誼.md`'s own `stand_in` points to this word (友誼), meaning 誼 is a bound morpheme that never appears independently outside this compound; 友's own `stand_in` points elsewhere (朋友). `kwin: false` per the AND-rule: `characters/友.md` is `kwin: false`, `characters/誼.md` is `kwin: true` → compound `false`.
- **Common-mispronunciation variant removed, not a genuine alternate reading**: `mandarin` previously listed `yǒuyì,yǒuyí` and `cantonese` listed `jau5 ji4,jau5 ji6`. Researched and confirmed yǒuyí is a very widespread mispronunciation (誼's phonetic component 宜 is yí, misleading speakers), but the standard, dictionary-correct reading is yǒuyì only — matching `characters/誼.md`'s own stored yì. Removed the second (incorrect) variant from both fields rather than presenting it as equally valid; the parallel Cantonese ji6 likely reflects the same confusion.
- **Wrong-word conflation corrected**: `vietnamese` previously held tình bạn, the everyday native Vietnamese word for "friendship" — but that is not a reading of 友誼 itself. Corrected to hữu nghị, the genuine, extremely common Sino-Vietnamese reading (verified via search, used constantly in diplomatic/official contexts, e.g. "Cầu Hữu Nghị," the Friendship Bridge).
