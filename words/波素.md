## Notes
### Form
波素 is [波](characters/波.md) “wave” + [素](characters/素.md) “element.” The suffix follows the rule for main-group elements.

### Motivation
The international name _polonium_ is derived from _Polonia_ “Poland,” named by Marie and Pierre Curie (1898) in reference to Marie Curie’s homeland. In Sinitic usage, “Poland” is [[波蘭]], a phonetic transcription. Selecting 波 as the element symbol is therefore a **reduction of a conventional place-name transcription**, not a semantic description.

### Type of formation
This is neither a direct semantic calque nor a pure phonetic loan. It is a **semantic–phonetic hybrid by abbreviation**:

- source: 波蘭 (phonetic for “Poland”)
- reduction: 波 (first graph retained as identifier)
- suffixation: 素 to mark elemental status

This parallels practices such as 英 for “England/English,” where the graph does not transparently encode the meaning “England” but functions as a conventional sign.

### Fit within the system
1. Maintains the no-Latin constraint.
2. Preserves a link to the international naming origin (Poland) via established CJKV transcription.
3. Avoids forced semantic reinterpretation, which would be difficult given the opaque Indo-European etymology.

### Comparative CJKV forms
- Modern Chinese: 钋 (phonetic)
- Japanese: ポロニウム (phonetic)
- Korean: 폴로늄 (phonetic)
- Vietnamese: poloni (phonetic)

All modern forms are phonetic; 波素 reintroduces a graph-based system using an existing toponymic convention.

### Clarity and ambiguity
- 波 “wave” is semantically unrelated to the element; the connection is conventional rather than descriptive.
- Within a chemical register, 波 is unambiguous if reserved uniquely for this element.
- Potential overlap with other 波-based abbreviations (e.g., 波斯 for Persia) is not problematic if those are not elements.

### Conclusion
波素 is a structurally consistent solution for a place-name-derived element whose Indo-European etymology offers no clear semantic handle. It leverages established CJKV transcription (波蘭) in the same way that 英 functions for England, maintaining internal coherence without resorting to phonetic spelling.
