## Notes
- [毎 (char)](../characters/毎%20(char).md) "every; each; distributive" + [時 (char)](../characters/時%20(char).md) "time; moment; season; occasion"

毎時 is the distributive temporal correlative — "each time; at every occasion, one instance at a time." As a 副用名詞 (adverbial noun), it distributes a predicate across each individual temporal occasion rather than asserting it as a universal constant: 毎時、確認を行う ("each time, perform a check"); 毎時この手順を繰り返す ("repeat this procedure each time"). The distributive 毎 marks individuated application — the predicate holds for each separate occasion — which is distinct from the collective universal 皆時 (always, at all times as a general truth).

Mandarin 每次 (měi cì, "each time, every occasion") is the most common counterpart — 次 (occasion, instance) functions as the temporal classifier where Dan'a'yo uses 時 (time). Japanese 毎回 (maikai, "each time/occasion") uses 毎 directly, making this one of the most morphologically transparent CJKV parallels: 毎回確認する ("check each time"), 毎回同じミスをする ("make the same mistake each time"). Korean 매번 (mae-beon, 毎番, "every time") is the standard everyday form — 매번 잊어버린다 ("I forget every single time"). The distributive meaning is consistent across all three.

毎時 must be carefully distinguished from 皆時 (always, collectively): 毎時 applies the predicate to each temporal occasion one by one — it is procedural, instructional, habitual — while 皆時 asserts the predicate holds universally across all time as a state. 毎時この薬を飲む ("take this medicine each time — e.g., with each meal") vs 皆時この薬を飲んでいる ("I am always taking this medicine — a constant state"). The former counts occasions distributively; the latter expresses a continuous universal truth.
