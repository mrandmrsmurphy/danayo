## Notes
- [毎 (char)](../characters/毎%20(char).md) "every; each; distributive" + [物 (char)](../characters/物%20(char).md) "thing; object; material result"

毎物 is the distributive correlative in the object column: it refers distributively to each individual thing, one by one. As in the action column (毎事), the character 毎 marks the distributive reading that individuates across all members of a set — "per thing, for each object, item by item." 毎物 functions as a 副用名詞 (adverbial noun) that can appear without a case particle to set the frame for the following predicate, on the pattern of 毎日 (every day), 毎月 (every month), 毎回 (every time): 毎物、丁寧に扱う ("handle each item with care").

In classical Chinese, 每物 (měi wù) appears as an adverbial quantifier meaning "for each thing, per item." Japanese 毎物 (まいぶつ) is not current vocabulary, but the 毎-prefix pattern is morphologically transparent. Mandarin colloquial equivalents are 每件东西 (per item, each thing) or 每一个 (each one), using the classifier construction that Dan'a'yo bypasses by treating 物 itself as the classifier-free nominal head.

The distributive/collective contrast central to the 毎/皆 column distinction operates in the 物 row as in all other rows: 毎物 individualises ("each thing, one at a time"), while 皆物 universalises collectively ("all things, the lot of them"). The same distinction appears in counting contexts — 毎物を確認する ("check each item") vs 皆物を確認する ("check all the things") — where the distributive reading implies sequential inspection and the collective reading implies a summary check of the totality.
