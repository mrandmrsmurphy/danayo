## Notes
- [草 (char)](../characters/草%20(char).md) "grass" + [花](../characters/花.md) "flower" — stand-in for [[花]], which cannot appear independently

草花 (grass-flower) names flowering plants collectively, with a herbaceous/humble connotation next to showier cultivated blooms — a genuine semantic contribution from 草, not just a legitimizing filler syllable for bounded 花.

Real conflation caught and fixed here: this page's cross-linguistic fields (`mandarin: huāduǒ`, `korean: 꽃`, native Japanese はな) previously described an entirely different word, 花朵 ("a flower, a blossom" — 花 + 朵, a flower-counting measure word), which is not a separate entry anywhere in this vault; the `aliases: [花朵]` field was almost certainly the root cause, the same mistaken-identity pattern as [[人等]]/[[我等]]/[[其人等]] earlier in this sweep. Corrected every field to describe 草花 itself: Mandarin cǎohuā (confirmed via Wiktionary, glossed as "showy/ornamental flowers," and — in a striking dialectal aside — also the Beijing-region card-game name for the clubs ♣ suit, alongside 梅花ㅡ"plum blossom" elsewhere). Japanese has two real readings: native くさばな (kusabana, "grass-flower," used for the kept-in-frontmatter value) and on'yomi そうか (sōka), both meaning "flowering plants" generally. Korean 초화 (chohwa) is the real Sino-Korean reading, alongside the everyday native 꽃 (kkot, "flower") which is what the old, wrong field actually held. Vietnamese thảo hoa is attested (also found in reversed order, 花草/hoa thảo, used interchangeably), alongside a fully native calque cỏ hoa.
