## Notes
- [可 (char)](../characters/可%20(char).md) "able to, may" + [以 (char)](../characters/以%20(char).md) "by means of, with" — Dan'a'yo's preposition for permission, distinct from the bare prefix [可](../characters/可%20(char).md) which marks ability/capacity. 可以 is one of the most common words in Mandarin, where it does the same double duty (能力 and 許可) that Dan'a'yo splits between the bare prefix and this preposition. `kwin: false` per the AND-rule: `characters/可 (char).md` is `kwin: false`, `characters/以 (char).md` is `kwin: true` → compound `false`.
- Attested across Mandarin and Cantonese, and — confirmed via search this iteration — as khả dĩ, a real if formal/literary Sino-Vietnamese word ("can be, is acceptable"). Japanese and Korean genuinely lack a corresponding word built from these characters; both instead use entirely native grammatical constructions for "can/may" (Japanese ～できる, Korean -을/ㄹ 수 있다), so `japanese`/`korean` were deliberately left blank rather than fabricated, the same honesty standard applied to [[不但]]/[[分之]] earlier this sweep.
