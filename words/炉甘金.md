## Notes
**Form.** 炉甘金 is [炉](characters/炉.md) “furnace” + [甘 (char)](characters/甘%20(char).md) “sweet” + [金 (char)](characters/金%20(char).md) “metal,” following the rule for transition elements, with a tri-character base.

**Etymological chain.** The international name _cadmium_ derives from Latin _cadmia_, from Greek καδμεία (_kadmeia_), the name for calamine/zinc ores. In Chinese mineral terminology, calamine is **炉甘石** (“furnace-calamine stone”), a well-attested traditional term.

The formation is:

- [[炉甘石]] (calamine)
- remove 石 (since we are naming an element, not a mineral)
- add 金 (metal suffix)  
    → 炉甘金

**Type of formation.**  
A **mineral-based semantic derivation**:

- element named after an ore
- ore rendered in classical Sinitic terminology
- element reconstructed from that term

This is particularly appropriate because cadmium is historically tied to zinc ores like calamine.

**Motivation.**

- Preserves the **historical material context** of the naming.
- Avoids the opaque Greek/Latin form, which does not transparently map into Sinitic.
- Uses an existing, meaningful compound rather than inventing a new abstraction.

**Fit within the system.**

1. Meaning-based, not phonetic.
2. Uses 金 correctly for a d-block element.
3. Accepts a tri-character structure where necessary.
4. Provides a distinctive grapheme (甘) for symbol use.

**Comparative CJKV forms.**

- Modern Chinese: 镉 (phonetic)
- Japanese: カドミウム (phonetic)
- Korean: 카드뮴 (phonetic)
- Vietnamese: cadimi (phonetic)

Traditional Chinese mineralogy (炉甘石) preserves the semantic layer that modern element names lost.

**Clarity and issues.**

- 炉甘 is not transparent to modern readers without some classical or technical background.
- However, within a controlled lexicon, it is precise and historically grounded.
- The mapping from mineral → element is conceptually coherent.

**Conclusion.**  
The logic is clear and well-founded. 炉甘金 is a strong example of using traditional mineral terminology to recover the etymological basis of an otherwise opaque Western name.
