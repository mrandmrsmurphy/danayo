# 単亜語入門（日本語話者向け）

## 一、単亜語とは何か

単亜語は、漢字を基礎とする分析型言語である。

特徴：
- 声調がない
- 一字一音節
- 連音・同化がない
- 動詞活用がない
- 基本語順は SVO（主語―動詞―目的語）
- 新字体を用いる

例：読、国、単


## 二、音節構造
単亜語はモーラではなく、**音節**を単位とする。

一音節は次の形をとる：

子音＋母音（＋子音）

各音節は独立して発音され、前後の音に影響されない。


## 三、声調はない

単亜語には声調の区別がない。

日本語のような高低アクセントとも無関係である。

すべての音節は同じ高さで発音してよい。

## 四、語末子音

### 1. -m / -n / -ng の区別

単亜語では
- -m
- -n
- -ng

は明確に区別される。

日本語の「ん」は一種類であるが、単亜語では三種類が独立した音として存在する。

したがって、すべてを同じ「ん」として処理しない。

### 2. -p / -t / -k について

単亜語には語末の
- -p
- -t
- -k

がある。

これは日本語の促音の「止め」に近い。

しかし、

- その後に母音は来ない
- 一音節で終わる
- 「隠れた母音」は存在しない

重要：

日本語では語末に子音だけで終わることはない。  
単亜語では可能である。

母音が消えるのではなく、最初から存在しない。

## 五、基本文型

単亜語の基本語順は SVO である。

日本語：  
私は本を読む。

単亜語：  
我 閲読 書本

助詞は不要である。

語順が文法関係を決める。

## 六、時体標識

単亜語では、動詞の前に時体を示す字を置く。

|字|機能|
|---|---|
|已|完了|
|未|未完了・否定完了|
|将|未来|
|中|進行|

例：

我 已 食  
我 中 食  
我 将 食  
我 未 食

動詞自体は変化しない。

## 七、格助詞

単亜語には位置固定の格字がある。

|単亜語|日本語対応|
|---|---|
|于|に|
|於|で／に|
|以|で|
|共|と|
|比|より|
|由|から|
|至|まで|
|似|のように|
|哉|よ|
|様|に|

これらは必ず定位置に置く。

## 八、主題文と「是」

単亜語では主題構文が可能である。

ただし、述語が名詞の場合は必ず「是」を用いる。

正：

此書 是 好書

省略してはならない。

## 九、文末助詞

単亜語にも文末助詞がある。

例：

也（強調）  
耶（軽い疑問）  
乎（確認）  
哉（感嘆）  
了（完了）

文末に置き、順序は固定である。

## 十、要点整理

1. 声調なし
    
2. 一字一音節
    
3. モーラではなく音節
    
4. 語末子音がある
    
5. 活用なし
    
6. SVO語順
    
7. 名詞述語には「是」
    

単亜語は、日本語と同様に漢字文化圏に属するが、  
構造はより規則的で単純である。
