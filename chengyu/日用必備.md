[Dan'a'yo Chengyu](chengyu/Dan'a'yo%20Chengyu.md)

>[!Tip] The motto of 常用字, the core character inventory of 単亜語 — see [[文法 - 03文字法]]

## Literal Meaning
- [日用](../words/日用.md) — daily use, everyday
- [必備](../words/必備.md) — essential, required, a must-have

Literally: "Daily use, always needed."

## Extended Meaning
The defining criterion for membership in 常用字 (the roughly 2,000-character core inventory of Dan'a'yo): a character belongs there if and only if it is needed for *daily* expression and *cannot be dispensed with*. The phrase names a double test rather than a single property — 日用 screens for frequency and ordinariness (a character used constantly, not a literary rarity), while 必備 screens for necessity (a character without which ordinary expression would be impossible, not merely convenient). A character that is common but replaceable, or essential but only in a narrow specialist register, fails one half of the test and belongs instead to 先進字 or 名字.

## Source and Origin
Coined within 単亜語 itself (2026-06-20) as the named motto for 常用字, the first and largest of the six character classes defined in [[文法 - 03文字法]] (the writing-system chapter). Built by directly joining two existing real CJKV words, [[日用]] ("daily use," as in 日用品) and [[必備]] ("essential, required," as in 必備条件) — the same construction method as [[文言継承]] (built from [[文言]] + [[継承]]). The pairing gives 常用字 the same kind of named, citable maxim that [[異体不容]] gives the variant-suppression principle and that [[文言現代]] gives the syntax chapter's overall design — a short, memorable four-character handle for a structural rule that the surrounding prose would otherwise only describe at length.

## Invocation
Invoke 日用必備 when explaining *why* a given character is (or is not) classed as 常用字 rather than 先進字 or 名字 — the test is always the conjunction of the two halves: is it used daily, and is it indispensable? Reach for it specifically when justifying the boundary of the core inventory, in the same register as [[異体不容]] justifies the variant-suppression rule.

## Example Sentences
1. なぜこの字が常用字なのか — 日用必備の原則による。日常的に使われ、かつ不可欠だからだ。
2. 이 글자가 상용자로 분류되는 이유는 일용필비다 — 매일 쓰이고, 없으면 안 되기 때문이다.
3. Not every common character makes the cut, and not every essential one does either — 常用字 requires both halves of 日用必備: daily *and* indispensable.

## Sentiment
Doctrinal and definitional, not evaluative — a statement of the membership criterion for 常用字, in the same register as [[異体不容]] and [[文言継承]]. Best invoked when explaining or defending the boundary of the core character inventory, rather than when discussing any single character's everyday use in isolation.
